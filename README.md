# Курсовая работа: Интеграция с API hh.ru и работа с вакансиями

Программа предназначена для работы с вакансиями с сайта [hh.ru],  включая:
- Получение данных через API.
- Фильтрацию и сортировку по зарплате.
- Сохранение в JSON-файл.
- Удобное взаимодействие с пользователем через консоль.

---

##  Возможности

- Поиск вакансий по ключевому слову (например, "Python").
- Показ топ-N вакансий по зарплате.
- Поиск по ключевым словам в описании.
- Добавление и удаление вакансий из файла.
- Сохранение результатов в формате JSON.

---

##  Технологии

- Python 3.10+
- `requests` — для работы с API.
- `json` — для хранения данных.
- `abc` — абстрактные классы.
- `pytest` — тестирование (по желанию).
